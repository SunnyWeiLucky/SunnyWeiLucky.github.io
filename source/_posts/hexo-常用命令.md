&nbsp;&nbsp;&nbsp;&nbsp;记录一下hexo经常使用的命令，发现自己好久没有hexo，命令都不记得了。

### 新建文章
```
hexo new "postname" (hexo n postname)
```

### 新建页面
```
hexo new page "pagename"
```

### 启动本地web服务
```
hexo server (hexo s)
本地网址：localhost:4000
```
### 生成静态文件
```
hexo generate (hexo g)
```

### 生成部署
```
hexo d -g
```

### 生成预览
```
hexo s -g
```

### 清理public文件夹
```
hexo clean 
```
